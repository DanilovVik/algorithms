### Run demo

- 环境要求：>= Node.js v7
- 如果 Node 版本较低，则可以自行选择使用 Babel 进行转换
- 直接运行各个文件夹下的 `js` 文件即可查看算法运行的结果
- 例如在项目根目录下 `node ./priority-queues/binary-heaps.js`
